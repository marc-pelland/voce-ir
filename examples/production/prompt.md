# Production Build Prompt

## What we're building

The voce-ir.xyz landing page — the official homepage for the Voce IR project.

## Sections

1. **Navigation bar** — Logo "Voce IR" on the left, links to Docs, Playground, GitHub on the right
2. **Hero section** — Large headline "The code is gone. The experience remains.", subtitle explaining what Voce IR is, two CTAs: "Get Started" (primary) and "View on GitHub" (secondary)
3. **Pipeline visualization** — Show the 4-step pipeline: Conversation → IR → Validate → Compile, with brief descriptions
4. **Feature grid** — 3-column grid with 6 features: Zero Runtime, 7 Targets, 46 Validation Rules, Accessible by Default, AI-Native, Production Ready
5. **Code example** — Show a minimal IR JSON snippet and the compiled HTML output side by side
6. **Getting started CTA** — Simple section with "Ready to build?" headline and install command
7. **Footer** — Copyright, links to docs, GitHub, license

## Design

- Dark theme (near-black background, white text, indigo accent #6366f1)
- Clean, minimal typography
- No images needed — text-only design
- Responsive: stacks to single column on mobile
